🚀 FastAPI REST API
📌 Descripción

API REST desarrollada con FastAPI para el manejo y exposición de datos mediante endpoints HTTP.
Este proyecto fue creado como prueba para implementar una arquitectura backend ligera, rápida y eficiente.

🧰 Tecnologías utilizadas

Python

FastAPI

Uvicorn

Pydantic

🔗 Características

⚡ Alta performance con FastAPI

📄 Validación automática de datos con Pydantic

📡 Endpoints REST

📘 Documentación automática con Swagger

📦 Endpoints principales


⚙️ Instalación y ejecución

Clonar repositorio:

git clone https://github.com/TU_USUARIO/fastapi-app.git

Entrar al proyecto:

cd fastapi-app

Crear entorno virtual:

python -m venv venv

Activar entorno virtual:

Windows:

venv\Scripts\activate

Mac/Linux:

source venv/bin/activate

Instalar dependencias:

pip install -r requirements.txt

Ejecutar servidor:

uvicorn main:app --reload
📘 Documentación automática

FastAPI genera documentación interactiva automáticamente:

Swagger UI:
http://localhost:8000/docs

ReDoc:
http://localhost:8000/redoc
